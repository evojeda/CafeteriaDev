☕ CafeteriaDev

Interface web inspirada em uma cafeteria moderna, desenvolvida utilizando HTML e CSS para praticar criação de layouts e estilização de páginas.

🔗 Acesse o projeto online:
https://evojeda.github.io/CafeteriaDev/

📌 Sobre o projeto

O CafeteriaDev é uma página web criada para simular o site de uma cafeteria, apresentando um layout simples e elegante.
O projeto foi desenvolvido com foco em prática de estruturação de páginas e design visual, utilizando apenas tecnologias fundamentais do front-end.
A interface apresenta elementos comuns em sites de cafeterias, como:
seção de apresentação;
destaque para produtos ou bebidas;
layout visual organizado e estilizado.

🧠 Tecnologias utilizadas

HTML5;
CSS3;
GitHub Pages (deploy do projeto).

🎯 Objetivo do projeto

Este projeto foi criado com o objetivo de praticar:
estruturação de páginas com HTML;
estilização com CSS;
organização de layout;
publicação de projetos na web com GitHub Pages.
